# dod

## Clone

```bash
git clone --recursive https://github.com/rishteam/dod.git
```

## Build

* Dependencies
  * SFML 2.5.1
  * ImGui-SFML
  * fmt
  * nlohmann-json

```bash
pacman -S - < package.txt
```

* Environment
  * msys2
  * cmake

```bash
./scripts/build.sh full run
```

* Generate documentation
  * `make doxygen-docs`
